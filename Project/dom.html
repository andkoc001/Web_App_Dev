DOM Manipulation

"The Document Object Model is a cross-platform and language-independent interface that treats an XML or HTML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree" (Wikipedia).

DOM Funtionality Demonstration

A few of the DOM functionalities are shown here below. The main focus of the demonstration is on accsess and change of the web page as well as the web page styling through the DOM. Additionally, examples of interaction from the user is shown.

Style modyfication

Here are two examples of changing website elements' style through DOM. The first button will cause the text style to become italic and uppercase. The second one will change the colours of the background and the text. The Reset button restores the original state.

Content modyfication

DOM manipulation allows also changing the content of the website. Again, two examples are provided. The first is activated by the left button, which results in changing the below sample text. The second button will evoke an image. Currently an empty element is used as a placeholder to maintain the web page structure. The Reset button restores the state as before changes.
