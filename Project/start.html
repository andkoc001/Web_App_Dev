DOM manipulation

In this section, a number of Document Object Model (DOM) examples are presented. These were divided into three groups: Style Modification, Content Modification, HTML elements animation and User Interaction. Further examples of the DOM manipulation are also presented in the data visualisation sections.

Data visualisation - data from an array

Data visualisation with the help of the D3.js library. The data is obtained from an embedded array (the array is defined inside the corresponding JavaScript file). The web page shows the visualisation of the planets of Solar System properties: radius, density and gravity. These properties are shown in a form of circles, where the size of the circle represents the value of the properties.

Data visualisation - data from a csv file

Data visualisation with the D3.js library application. This time, the data is imported from csv files. Two data sets are used: private car sales in Ireland in years 2010 to 2019 and overall life satisfaction in EU countries. The user is offered to apply some modification to the data visualisation and the way it is presented (e.g. animation speed).
